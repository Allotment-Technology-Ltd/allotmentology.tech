# Product screenshots (optional)

Drop PNG or WebP files here to replace the built-in SVG vignettes on the marketing site.

| File           | Used for        |
| -------------- | --------------- |
| `restormel.png` | Restormel Keys |
| `sophia.png`    | SOPHIA         |
| `plot.png`      | PLOT           |

Recommended: ~1200px wide, 16:10 or similar, cropped UI (not full browser chrome unless intentional).

After adding files, set `screenshotSrc` on each product in `web/src/lib/public-site/content.ts` (paths like `/site/products/restormel.png`).
